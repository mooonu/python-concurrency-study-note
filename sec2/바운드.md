## 바운드
### Code
```python
import requests  
  
def io_bound_func():  
    result = requests.get("https://google.com")  
    return result  
  
if __name__ == "__main__":  
  
    result = io_bound_func()  
    result = io_bound_func()  
    result = io_bound_func()  
    result = io_bound_func()  
    result = io_bound_func()  
  
    print(result)
```

### Result
`<Response [200]>`

### Lesson
- CPU Bound (CPU 바운드)
	- CPU의 **연산 처리 속도**가 병목되는 경우
	- CPU가 계속 바쁘게 **계산 작업을 수행하는 동안 프로그램이 기다리는 상황**
		- 복잡한 수학 계산, 이미지 처리, 동영상 인코딩, 암호화/복호화, 데이터 압축 등
- I/O Bound (I/O 바운드)
	- 입출력 작업이 병목이 되는 경우
	- CPU는 대기하며 **I/O 작업이 완료되기를 기다리는 상황**
	    - 네트워크 요청 (API 호출, 웹 스크래핑), 사용자 입력 대기
 - 블록킹
	- CPU 나 I/O 바운드에 의해 코드가 멈추게 되는 현상(코드가 느려짐)
		- 위 코드애서는 5번의 블록킹이 일어남
- 논블록킹
	- 블록킹이 일어나지 않는 것 (바운드에 의해 코드가 멈추지 않는 것)
